GitHub

Site: https://github.com/

Repositórios e commits
1 - Crie uma conta 2 - Crie um repositorio (letras minúsculas de preferência)
3 - Clique em Clonar o repositório
4 - Entre no diretório local do seu projeto
5 - Num terminal digite o comando: git clone endereco_copiado
6 - Dentro da pasta desejada:

    git config user.email "email@email.com"
    git config user.name "Seu Nome"
    git add .
    git commit -m "Commit ajustando pastas no repositório"
    git push

Templates Literais:
Usado para não precisar concatenar variáveis com strings com sinal +.
Sintaxe: `Minha variável é: ${nome_da_variavel}`;
Detalhe: Usa-se acento (`) e não aspas duplas (") ou simples (').
